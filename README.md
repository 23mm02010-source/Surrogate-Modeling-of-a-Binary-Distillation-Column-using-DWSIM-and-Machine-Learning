# Surrogate-Modeling-of-a-Binary-Distillation-Column-using-DWSIM-and-Machine-Learning
This project focuses on developing a surrogate model for a binary distillation column (Benzene–Toluene system) using data generated from simulation and machine learning techniques.  The surrogate model predicts:  Distillate purity (xD) Bottoms purity (xB) Condenser duty (QC) Reboiler duty (QR)
